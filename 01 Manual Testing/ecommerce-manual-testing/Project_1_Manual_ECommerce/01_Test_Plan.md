# Test Plan: E-commerce Web Application

## 1. Objective
- เพื่อทดสอบฟังก์ชันหลักของระบบ E-commerce ได้แก่ Login, Cart, Checkout และ Payment
- เพื่อระบุความเสี่ยงและกำหนดขอบเขตการทดสอบ (In-scope / Out-of-scope)

## 2. Target Application
- URL: Localhost (demo-app/index.html)

## 3. Test Strategy & Techniques
- Black-box Testing
- Boundary Value Analysis (BVA) สำหรับช่องกรอกตัวเลข/ราคา
- Equivalence Partitioning (EP) สำหรับช่องกรอกข้อมูล
- Exploratory Testing สำหรับหา Edge cases

## 4. Environment
- OS: Windows 11
- Browser: Chrome (Version 120.0), Firefox (Version 121.0)
